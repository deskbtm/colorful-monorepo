# Colorful Monorepo

这个插件不是免费的，请看 [价格](#价格)。当然这是自愿的。

`Colorful Monorepo` 可以在一定程度上帮助你提高 Monorepo 项目的开发效率。

## 价格

如果你获得了一个良好的体验请支付 1 ￥。

## 使用

### 1. 设置 Workspace

支持 `npm`,`yarn`, `pnpm`, `lerna`, `rush`。
![png1](./assets/Peek%202022-05-03%2021-16.gif)
当然也可以右击`As Workspace`手动添加，不包含`package.json`的文件夹。
